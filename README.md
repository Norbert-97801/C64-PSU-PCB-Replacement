# C64-PSU-PCB-Replacement

## PSU PCB Replacement for the Commodore C64 PSU 902503-06


![C64 Transformer 902503-06](images/C64%20Transformator%20902503-06.jpg "C64 Transformer 902503-06")

<img align="left" src="images/C64-PSU-PCB-Rev.C-2023-07-26.jpg" title="New PCB for the C64 PSU 902503-06" alt="New PCB for the C64 PSU 902503-06"> Here is my soloution for the C64 PSU probelm when the PSU is going to fail. I've developed a PSU PCB to replace the original PSU PCB. The build in 7805 will be used as a mounting point to install the new PCB in the Transformer housing. The new PCB will be using a adjustable DC/DC Converter to get the 5.1V. The Diode brigde rectifier are build with 3A schottky diodes.
